# Hermes Agent Knowledge Base

> 💡 **一次教，永远记住** — 这是为 Hermes Agent 准备的知识库，让新创建的 Agent 无需从头学习。

## 这是什么？

这是 Hermes AI Agent（jackwan317 的助手）在与用户交互过程中积累的：
- 🔧 **工具配置** — 飞书 API、Bitable、文档 ID
- 🧠 **工作流** — 每日复盘、感悟记录、想法管理
- 🎯 **自定义技能** — 对抗性 Idea 评审系统
- 👤 **用户画像** — 偏好、习惯、技术栈

新创建的 Hermes Agent 读取这个仓库后，就能无缝接续工作，无需用户重复教学。

## 新 Agent 如何使用？

👉 从 **[HERMES.md](HERMES.md)** 开始，按 4 步操作：

1. 加载用户画像
2. 学习持久记忆
3. 了解自定义技能
4. 安装技能文件
